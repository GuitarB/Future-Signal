# Future Signal

Future Signal is a futuristic web experience where users ask bold questions about their future, ideas, plans, or possible next moves and receive a dramatic signal-style forecast.

## Concept

This project is designed to feel fast, cinematic, and futuristic. It is built as a simple static website so it can be deployed easily with GitHub and Cloudflare Pages.

Examples of prompts:

- What happens if I start a business this year?
- What if I reinvent my life in the next 90 days?
- What if I build something people cannot stop talking about?
- What kind of future am I creating right now?
